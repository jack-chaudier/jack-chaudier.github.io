jack-chaudier/ocean-linux ↗

Ocean Microkernel

Memory, scheduling, and userspace isolation from scratch.

An x86_64 microkernel focused on memory management, scheduling, and userspace isolation — built because I wanted to know exactly what was happening underneath my programs.

  • Boot and runtime fundamentals: Limine boot path, GDT / IDT setup, fast interrupt handling, and a SYSCALL / SYSRET system call interface.
  • Kernel memory systems including a buddy allocator, 4-level paging, and slab allocation with explicit ownership boundaries.
  • Capability-based IPC and userspace server boundaries that keep policy outside kernel space while preserving performance.
